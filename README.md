# 📊 HR Analytics Dashboard | Power BI Project

## 🌟 Project Overview
This **HR Analytics Dashboard** was developed to help organizations make data-driven decisions regarding their workforce. By analyzing key employee metrics, the dashboard provides actionable insights into employee attrition, satisfaction levels, and demographic distributions, allowing leadership to identify risks and optimize retention strategies.

---

## 🎯 Key Metrics & KPIs
The dashboard tracks and visualizes critical HR indicators:
*   **Workforce Density:** Total employee count segmented by department and job role.
*   **Attrition Analysis:** Identifying root causes of employee turnover (e.g., job satisfaction, distance from home, or monthly income).
*   **Demographic Insights:** Gender diversity, age distribution, and educational background.
*   **Performance Tracking:** Correlation between years of experience, performance ratings, and salary hikes.
*   **Tenure & Retention:** Average years at company and time since last promotion.

---

## 🛠️ Technology Stack
*   **Business Intelligence Tool:** Microsoft Power BI Desktop
*   **Data Modeling:** Star Schema / Snowflake Schema
*   **Logic Engine:** DAX (Data Analysis Expressions) for custom measures
*   **Data Transformation:** Power Query (M Language)

---

## 💡 Key Insights Captured
1.  **High-Risk Departments:** identified specific departments with higher-than-average attrition rates.
2.  **Satisfaction Correlation:** Discovered the impact of job involvement and work-life balance on employee longevity.
3.  **Salary Benchmarking:** Visualized the gap between monthly income and job roles to ensure competitive compensation.

---

## 🚀 How to View the Dashboard
1.  **Download:** Clone this repository or download the `HR_DASHBOARD.pbix` file.
2.  **Install:** Ensure you have [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed.
3.  **Launch:** Open the `.pbix` file to interact with the visualizations.
